# Programa Luizalabs - Back-end com Python da plataforma Digital Innovation One

Este repositório contém atividades práticas realizadas durante o bootcamp de python da plataforma **DIO**.

Possui o objetivo de expor meu aprendizado, progressão e envolvimento com o programa. Revisarei meu conhecimento sobre a linguagem e conhecerei novos conceitos e ferramentas.

## Conteúdo do programa

O programa possui os seguintes módulos:

- Primeiros Passos com Python e Versionamento de Código
- Dominando Funções e Boas Práticas em Python
- Programação Orientada a Objetos com Python
- Fundamentos de APIs REST com FastAPI
- Integrando Python com Banco de Dados Relacionais
- Evoluindo com APIs Assíncronas com FastAPI
- Aumentando a Produtividade com IA e Versionamento de Código
